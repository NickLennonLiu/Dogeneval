命令功能

在CSLB上查询业务、CSLB和VNRS根据VPN名称分配的VPN索引。完成VPN配置后，可以通过这个命令检查是否有VPN分配失败。

注意事项

无

操作用户权限

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

参数说明

参数标识

参数名称

参数说明

SRVVNFCID

业务VNFC ID

可选必选说明：可选参数

参数含义：该参数用于指定业务VNFC的唯一标识，通过在窗口下执行LST NODE获得，所得NODEID即为服务VNFC ID。

数据来源：全网规划

取值范围：整数类型，取值范围为0~4294967295。

默认值：无

VPNNAME

VPN名称

可选必选说明：可选参数

参数含义：业务配置的VPN名称。

数据来源：全网规划

取值范围：字符串类型，输入长度范围为0~31。

默认值：无

VPNQUERYMODE

VPN查询模式

可选必选说明：可选参数

参数含义：设置VPN查询模式。

数据来源：本端规划

取值范围：枚举类型
  • VALID：查询有效的VPN。
  • INVALID：查询无效的VPN。
  • ALL：查询所有VPN。

默认值：INVALID

使用实例

  • 查询CSLB上的VPN信息。

    DSP LBVPN: VPNQUERYMODE=ALL;

    %%DSP LBVPN: VPNQUERYMODE=ALL;%%
RETCODE = 0  操作成功

操作结果如下：
--------------
业务VNFCID  VPN名称   业务分配的VPN索引  CSLB分配的VPN索引  VNRS分配的VPN索引

2504        _public_  1                  0                  1                  
129         _public_  1                  0                  1                  
(结果个数 = 3)

---    END

输出结果说明

输出项名称

输出项解释

业务VNFC ID

参数含义：指定业务VNFC的唯一标识。

取值范围：整数类型，取值范围为0~4294967295。

VPN名称

参数含义：业务配置的VPN名称。

取值范围：字符串类型，输入长度范围为0~31。

业务分配的VPN索引

参数含义：业务根据VPN名称分配的VPN索引。

取值范围：0~63字符

CSLB分配的VPN索引

CSLB根据VPN名称分配的VPN索引，如果为4294967295表示无效值，需要联系业务VNFC查看VPN配置插件返回的错误码定位原因。

取值范围：0~63字符

VNRS分配的VPN索引

VNRS根据VPN名称分配的VPN索引，如果为4294967295表示无效值，检查VNRS上是否配置了这个VPN。

取值范围：0~63字符
