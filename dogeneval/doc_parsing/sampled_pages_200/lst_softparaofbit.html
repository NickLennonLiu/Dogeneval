命令功能

适用网元：SGSN、MME

该命令用于查询软件参数表，查询结果以二进制形式输出。同时，该命令也支持查询指定软件参数中某个比特位的值。

注意事项

无。

本地用户权限

manage-ug；system-ug；monitor-ug

网管用户权限

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

参数说明

参数标识

参数名称

参数说明

DT

参数类型

可选必选说明：必选参数

参数含义：软件参数的数据类型。

取值范围：
  • “BYTE(单字节类型)”
  • “DWORD(无符号整数类型)”
  • “BYTE_EX(扩展单字节类型)”
  • “DWORD_EX(扩展无符号整数类型)”
  • “BYTE_EX_B(扩展单字节类型B)”
  • “DWORD_EX_B(扩展无符号整数类型B)”

默认值：无。

PARANUM

参数索引

可选必选说明：可选参数

参数含义：软件参数索引。

取值范围：1~512

配置原则：
  • 当参数类型取值为“BYTE”时，参数索引的取值范围是“1~240”。
  • 当参数类型取值为“DWORD”时，参数索引的取值范围是“1~60”。
  • 当参数类型取值为“BYTE_EX”时，参数索引的取值范围是“1~224”。
  • 当参数类型取值为“DWORD_EX”时，参数索引的取值范围是“1~56”。
  • 当参数类型取值为“BYTE_EX_B”时，参数索引的取值范围是“1~464”。
  • 当参数类型取值为“DWORD_EX_B”时，参数索引的取值范围是“1~116”。

默认值：无。

POSITION

比特位

可选必选说明：可选参数

参数含义：参数比特位的位置。

取值范围：1~32

配置原则：
  • 对于软件参数，右边的第一位记为比特1。例如，参数值 = 0000 0001，表示比特1的参数值为1。
  • 当参数类型取值为“BYTE”，“BYTE_EX”或“BYTE_EX_B”时，比特位的取值范围是“1~8”。
  • 当参数类型取值为“DWORD”，“DWORD_EX”或“DWORD_EX_B”时，比特位的取值范围是“1~32”。

默认值：无。

使用实例

查询当前参数索引为1，比特位为1的单字节类型软参的设置情况：

LST SOFTPARAOFBIT: DT=BYTE, PARANUM=1, POSITION=1;

%%LST SOFTPARAOFBIT: DT=BYTE, PARANUM=1, POSITION=1;%%

RETCODE = 0  操作成功。
查询结果如下
--------------
参数类型  =  单字节类型
参数索引  =  1
  比特位  =  1
  参数值  =  0
(结果个数 = 1)
---    END

查询当前参数索引为空，比特位为空的无符号整数类型软参的设置情况：

LST SOFTPARAOFBIT: DT=DWORD;

RETCODE = 0  操作成功。
