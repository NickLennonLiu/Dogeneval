命令功能

该命令是高危命令，操作不当可能会导致业务受损，请谨慎使用并联系华为技术支持协助操作。

该命令用于设置SDR服务的相关参数。SDR主要功能是为微服务间提供分布式透明通信。

注意事项

  • 该命令执行后立即生效。
  • SDR参数ID为0的功能该版本不支持。
  • SDR参数ID为4的功能，其对应的PARAVALUE1表示Fabric平面可靠传输的校验和开关。修改PARAVALUE1的值后，请全量复位POD。
  • 默认使用系统初始值，如需修改，请联系华为技术支持协助操作。
  • 系统部署完成后，已经存在初始记录，参数的初始记录值如下表：

参数说明

参数标识 参数名称 参数说明

PARAID

参数ID

可选必选说明：必选参数

参数含义：该参数用于指定SDR命令标识。

数据来源：本端规划

取值范围：整数类型，取值范围是0~7。

默认值：无。执行命令并不输入该参数时，系统默认为参数的初始设置值。

配置原则：无

PARAVALUE1

第一个参数值

可选必选说明：必选参数

参数含义：该参数用于SDR命令。

当SDR参数ID为0时，该参数值用于SDR可靠传输开关控制，0表示关闭base和fabric可靠传输开关，1表示只开启可靠base传输开关，2表示只开启可靠fabric传输开关，3表示开启base和fabric可靠传输开关。

当SDR参数ID为1时，该参数值用于Fabric链路的故障检测（告警）和业务隔离能力功能开关，0表示关闭，1表示开启。

当SDR参数ID为2时，该参数值用于DTP功能开关，0表示关闭，1表示开启。

当SDR参数ID为3时，该参数值用于SDRS内多线程消息处理转组件线程消息处理功能开关，0表示关闭，1表示开启。

当SDR参数ID为4时，该参数值用于Fabric可靠传输校验和开关，0表示关闭，1表示开启。

当SDR参数ID为5时，该参数值用于SDR话统数据上报开关，0表示关闭，1表示开启。

当SDR参数ID为6时，该参数值用于SDRS与SDRC之间的策略核查开关，0表示关闭，1表示开启。

当SDR参数ID为7时，该参数值用于SDR pbuf流控功能开关，0表示关闭，1表示开启。

数据来源：本端规划

取值范围：整数类型，取值范围是0~4294967295。

默认值：无。执行命令并不输入该参数时，系统默认为参数的初始设置值。

配置原则：无

PARAVALUE2

第二个参数值

可选必选说明：可选参数

参数含义：该参数用于SDR命令值扩展，类型是字符串。当SDR参数ID为6时，该参数值用于表示SDRS与SDRC之间的策略核查周期，单位为分钟。

当SDR参数ID为7时，该参数值用于表示pbuf流控功能起控阈值和pbuf使用率刷新周期，刷新周期单位为毫秒。

数据来源：本端规划

取值范围：字符串类型，输入长度范围是0~256。当PARAID的取值为6且PARAVALUE1取值为1时，请按照无符号整数类型的数值进行输入，有效数值范围1~4294967295，输入其他内容则该命令不生效。

默认值：无。执行命令并不输入该参数时，系统默认为参数的初始设置值。

配置原则：

当PARAID的取值为6且PARAVALUE1取值为1时，该参数为必选，若不输入，则该命令不生效。

使用实例

通过如下命令开启SDRS与SDRC之间的策略核查功能：

SET SDRPARAMS: PARAID=6, PARAVALUE1=1, PARAVALUE2="10";
