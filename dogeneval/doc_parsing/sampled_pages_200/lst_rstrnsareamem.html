查询TA级限制切片所属跟踪区成员（LST RSTRNSAREAMEM）

  • 命令功能
  • 注意事项
  • 参数说明
  • 使用实例
  • 输出结果说明

命令功能

适用NF：AMF

该命令用于查询指定TA级限制切片所属区域或者系统中配置的所有TA级限制切片所属区域的跟踪区成员。

注意事项

无

操作用户权限

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

参数说明

参数标识 参数名称 参数说明

AREACODE

区域编码

可选必选说明：可选参数

参数含义：该参数用于标识TA级限制切片所属的某区域范围。

数据来源：全网规划

取值范围：字符串类型，输入长度范围是1~128。该参数依赖于ADD RSTRNSAREACODE命令中的AREACODE参数。

默认值：无

配置原则：无

MCC

移动国家码

可选必选说明：可选参数

参数含义：该参数用于表示组成TA级限制切片所属区域的位置区成员的移动国家码，与无线接入网的MCC一致。

数据来源：全网规划

取值范围：字符串类型，输入长度是3。只允许输入十进制数字（0-9）。

默认值：无

配置原则：无

MNC

移动网号

可选必选说明：可选参数

参数含义：该参数用于表示组成TA级限制切片所属区域的位置区成员的移动网号，与无线接入网的MNC一致。

数据来源：全网规划

取值范围：字符串类型，输入长度范围是2~3。只允许输入十进制数字（0-9）。

默认值：无

配置原则：

两条记录间的MCC相同时，2位MNC与3位MNC的前两个数字不能相同。

BGNTAC

跟踪区编码起始值

可选必选说明：可选参数

参数含义：该参数用于表示跟踪区编码的起始值。

数据来源：全网规划

取值范围：字符串类型，输入长度是6。TAC编码为16进制数，按照字符串格式输入，字符串长度为6，只能由数字（0-9），字母（A-F、a-f）组成。字母大小写不敏感。

默认值：无

配置原则：无

使用实例

  • 查询编码为“jq001.pd006.sh.mcc123.mnc45”的TA级限制切片所属区域成员信息，执行如下命令： 
    %%LST RSTRNSAREAMEM: AREACODE="jq001.pd006.sh.mcc123.mnc45";%%
RETCODE = 0  操作成功

结果如下
--------
        区域编码  =  jq001.pd006.sh.mcc123.mnc45
      移动国家码  =  123
        移动网号  =  45
跟踪区编码起始值  =  111111
跟踪区编码结束值  =  111111
        描述信息  =  NULL
(结果个数  = 1)

---    END
  • 查询所有的TA级限制切片所属区域成员信息，执行如下命令： 
    %%LST RSTRNSAREAMEM:;%%
RETCODE = 0  操作成功

结果如下
--------
        区域编码  =  jq001.pd006.sh.mcc123.mnc45
      移动国家码  =  460
        移动网号  =  121
跟踪区编码起始值  =  111111
跟踪区编码结束值  =  111111
        描述信息  =  NULL
(结果个数  = 1)

---    END

输出结果说明

输出项名称 输出项解释

区域编码

该参数用于标识TA级限制切片所属的某区域范围。

移动国家码

该参数用于表示组成TA级限制切片所属区域的位置区成员的移动国家码，与无线接入网的MCC一致。

移动网号

该参数用于表示组成TA级限制切片所属区域的位置区成员的移动网号，与无线接入网的MNC一致。

跟踪区编码起始值

该参数用于表示跟踪区编码的起始值。

跟踪区编码结束值

该参数用于表示跟踪区编码的结束值。

描述信息

该参数是对区域成员的描述信息，在运维中起到助记的作用。
