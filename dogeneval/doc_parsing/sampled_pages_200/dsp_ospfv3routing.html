查询OSPFv3路由信息（DSP OSPFV3ROUTING）

命令功能

该命令用于显示OSPFv3路由信息。

注意事项

无。

操作用户权限

G_1，管理员级别命令组；G_2，操作员级别命令组；G_3，用户级别命令组

参数说明

参数标识

参数名称

参数说明

PROCESSID

OSPFv3进程号

可选必选说明：可选参数

参数含义：OSPFv3进程号。

数据来源：全网规划

取值范围：整数类型，取值范围为1～4294967295。

默认值：无

DESTIPV6ADDR

目的IPv6地址

可选必选说明：可选参数

参数含义：目的IPv6地址。

数据来源：本端规划

取值范围：IPv6地址类型。

默认值：无

IPV6MASKLEN

前缀长度

可选必选说明：可选参数

参数含义：前缀长度。

数据来源：本端规划

取值范围：整数类型，取值范围为0～128。

默认值：无

NEXTHOPIPV6ADDR

下一跳IPv6地址

可选必选说明：可选参数

参数含义：下一跳IPv6地址。

数据来源：本端规划

取值范围：IPv6地址类型。

默认值：无

IFNAME

接口名称

可选必选说明：可选参数

参数含义：接口名称。

数据来源：本端规划

取值范围：字符串类型，输入长度范围为1～63。

默认值：无

配置原则：请使用LST INTERFACE命令查看可用接口。

使用实例

显示设备OSPFv3进程的所有路由信息：

DSP OSPFV3ROUTING:;
RETCODE = 0  操作成功。

结果如下
------------------------
             OSPFv3进程号  =  1000
               路径Cost值  =  1
                 路径类型  =  directly connected
                   优先级  =  Low
               下一跳数目  =  1
             目的IPv6地址  =  2001:db8::
                 前缀长度  =  64
           下一跳IPv6地址  =  2001:db8::1
                 接口名称  =  Ethernet64/0/7
                 显示标志  =  Added to URT6
                 备份接口  =  NULL
       备份下一跳IPv6地址  =  ::
                 备份类型  =  None
(结果个数 = 1)
---    END

输出结果说明

输出项名称

输出项解释

OSPFv3进程号

该参数指定用户创建的OSPFv3进程号。

路径Cost值

该参数指定目的路由路径Cost值。

路径类型

该参数指定目的路由路径类型。

优先级

该参数指定目的路由的优先级。

下一跳数目

该参数指定目的路由可用的IPv6地址下一跳数目。

目的IPv6地址

该参数指定目的路由IPv6地址。

前缀长度

该参数指定目的路由前缀长度。

下一跳IPv6地址

该参数指定目的路由下一跳IPv6地址。

接口名称

该参数指定目的路由的出接口名称。

显示标志

该参数指定路由信息标记。

备份接口

该参数指定目的路由的备份出接口名称。

备份下一跳IPv6地址

该参数指定目的路由的备份下一跳IPv6地址。

备份类型

该参数指定目的路由备份路径类型。
