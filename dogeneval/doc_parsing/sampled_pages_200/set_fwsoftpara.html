设置Service Fabric软参（SET FWSOFTPARA）

  • 命令功能
  • 注意事项
  • 参数说明
  • 使用实例

命令功能

该命令用于设置Service Fabric软件参数。

注意事项

  • 该命令执行后立即生效。
  • 软参索引取值区间：1-2048。

操作用户权限

G_1，管理员级别命令组；G_2，操作员级别命令组

参数说明

参数标识 参数名称 参数说明

PARAMETERSTYPE

参数类型

可选必选说明：必选参数

参数含义：该参数表示软件参数的类型。

数据来源：本端规划

取值范围：

  • DWORD（双字）
  • DWORD_EX（扩展双字）

默认值：无。执行命令并不输入该参数时，系统默认为参数的初始设置值。

配置原则：无

DWORDINDEX

DWORD索引

可选必选说明：该参数在"PARAMETERSTYPE"配置为"DWORD"时为条件必选参数。

参数含义：该参数表示Dword类型软件参数的索引。

数据来源：本端规划

取值范围：整数类型，取值范围是1~2048。

默认值：无。执行命令并不输入该参数时，系统默认为参数的初始设置值。

配置原则：无

DWORDVALUE

DWORD取值

可选必选说明：该参数在"PARAMETERSTYPE"配置为"DWORD"时为条件必选参数。

参数含义：该参数表示Dword类型软件参数的值。

数据来源：本端规划

取值范围：整数类型，取值范围是0~4294967295。

默认值：无。执行命令并不输入该参数时，系统默认为参数的初始设置值。

配置原则：无

EXTENDDWORDINDEX

扩展DWORD索引

可选必选说明：该参数在"PARAMETERSTYPE"配置为"DWORD_EX"时为条件必选参数。

参数含义：该参数表示Dword extend类型软件参数的索引。

数据来源：本端规划

取值范围：整数类型，取值范围是1~2048。

默认值：无。执行命令并不输入该参数时，系统默认为参数的初始设置值。

配置原则：无

EXTENDDWORDVALUE

扩展DWORD取值

可选必选说明：该参数在"PARAMETERSTYPE"配置为"DWORD_EX"时为条件必选参数。

参数含义：该参数表示Dword extend类型软件参数的值。

数据来源：本端规划

取值范围：整数类型，取值范围是0~4294967295。

默认值：无。执行命令并不输入该参数时，系统默认为参数的初始设置值。

配置原则：无

使用实例

设置软参值

SET FWSOFTPARA: PARAMETERSTYPE=DWORD, DWORDINDEX=20, DWORDVALUE=20;
