命令功能

适用网元：SGSN

该命令用于设置计费Ga接口参数，包括SGSN生成话单的协议版本等参数。

注意事项

  • 系统初次运行时，会执行系统初始设置值。
  • 如果设置“CDR重发间隔（ITVRES）”、“硬盘操作失败告警次数门限（HDDETH）”、“硬盘空间不足门限（HDDFSL）”、“重定向帧最大占用率（REDICTFRMMAXOCCRATE）”，则只有在CDP进程重启之后才生效。
  • 如果设置“GPRS CDR协议版本（GCR）”、“UMTS CDR协议版本（UCR）”、“R98 CDR版本（CHGVER98）”、“R99 CDR版本（CHGVER99）”、“R4 CDR版本（CHGVER4）”、“R5 CDR版本（CHGVER5）”、“R6 CDR版本（CHGVER6）”、“R7 CDR版本（CHGVER7）”、“R9 CDR版本（CHGVER9）”，则只有重新启动SPP和UPP进程之后才生效。

本地用户权限

manage-ug；system-ug

网管用户权限

G_1，管理员级别命令组；G_2，操作员级别命令组

ITVRES

CDR重发间隔（s）

可选必选说明：可选参数

参数含义：该参数是CDP进程发送话单给CG之后，如果在该时间间隔之内没有收到CG的应答，则会再次重发。

数据来源：整网规划

取值范围：1～10

系统初始设置值：3

说明：

该参数修改后，必须重新启动CDP进程之后才能生效。

RSNUM

CDR重发次数

可选必选说明：可选参数

参数含义：该参数是CDP进程发送话单给CG的最大重发次数。

数据来源：整网规划

取值范围：1～10

系统初始设置值：3

说明：

该参数配置后立即生效，无需重新启动CDP进程。

HDDETH

硬盘操作失败告警次数门限

可选必选说明：可选参数

参数含义：该参数是CDP进程最大连续硬盘操作失败次数。如果超过该门限值，CDP进程触发计费业务硬盘操作失败告警。

数据来源：整网规划

取值范围：1～10

系统初始设置值：5

说明：

该参数修改后，必须重新启动CDP进程之后才能生效。

HDDFSL

硬盘空间不足门限（%）

可选必选说明：可选参数

参数含义：该参数是计费业务硬盘空间不足门限配置值。如果已使用空间与总空间的比值超过该门限值，CDP进程将在主备之间切换话单存储的硬盘。当两个硬盘空间都不足时，覆盖硬盘上话单文件若配置为不覆盖，则话单将丢失，若配置为覆盖，则删除硬盘旧话单腾出硬盘空间保存新话单。

数据来源：整网规划

取值范围：50～90

系统初始设置值：70

说明：

该参数修改后，必须重新启动CDP进程之后才能生效。

REDICTFRMMAXOCCRATE

重定向帧最大占用率（%）

可选必选说明：可选参数

参数含义：该参数用于指定重定向帧的最大占用率。重定向帧是指当CG故障了重新指定另一个CG时发出的帧。

数据来源：整网规划

取值范围：20～80

系统初始设置值：80

发送消息UDP校验

可选必选说明：可选参数

参数含义：该参数为Ga接口为UDP连接时发送消息是否进行UDP校验的开关。

数据来源：整网规划

取值范围：
  • “DISABLE(不生效)”
  • “ENABLE(生效)”

系统初始设置值：ENABLE(生效)

GAREVUDPCHECK

接收消息UDP校验

可选必选说明：可选参数

参数含义：该参数为Ga接口为UDP连接时接收消息是否进行UDP校验的开关。UDP校验指话单在GA口进行数据包传送时进行的包数据的校验检查。

数据来源：整网规划

取值范围：
  • “DISABLE(不生效)”
  • “ENABLE(生效)”

系统初始设置值：ENABLE(生效)

CDROVERWRITE

覆盖硬盘上话单文件

可选必选说明：可选参数

参数含义：该参数用于控制是否覆盖硬盘文件。开启硬盘覆盖功能时应保证硬盘上不存在话单文件（通过DSP CHGFILE查询），否则该功能删除的话单文件不能保证为最旧的文件。为了保证在开启硬盘覆盖功能之前，且已经存在硬盘上的话单删除完之前，不会删除在开启功能后新生成的话单，需要复位备CDP进程后倒换主备CDP进程。

数据来源：整网规划

取值范围：
  • “NO(不覆盖)”
  • “YES(覆盖)”

系统初始设置值：NO(不覆盖)

说明：

该参数配置后立即生效。

IPPLCY

IP地址选择策略

可选必选说明：可选参数

参数含义：该参数表示Ga接口IP地址策略。

数据来源：整网规划

取值范围：枚举类型。在IPv4网络中，配置成“IPV4(仅使用IPv4地址)”；在IPv4向IPv6改造过程中，配置成“IPV6PRE(优先使用IPv6地址)”；IPv6网络改造完成后，配置成“IPV6(仅使用IPv6地址)”。
  • IPV4：仅使用IPv4地址。
  • IPV6：仅使用IPv6地址。
  • IPV6PRE：优先使用IPv6地址。

系统初始设置值：IPV4

使用实例

设置一条计费Ga接口参数，“GPRS CDR协议版本”为“R99”，“UMTS CDR协议版本”为“R4”，“R98 CDR版本”为“GSM1215V760”，“R4 CDR版本”为“GPP32215V440”，“CDR重发间隔(s)”为4s，“CDR重发次数”为4，“硬盘操作失败告警次数门限”为85，“覆盖硬盘上话单文件”为“NO”：

SET CHGGA: GCR=R99, UCR=R4, CHGVER98=GSM1215V760, CHGVER4=GPP32215V440, ITVRES=4, RSNUM=4, HDDETH=4, HDDFSL=85, CDROVERWRITE=NO;
